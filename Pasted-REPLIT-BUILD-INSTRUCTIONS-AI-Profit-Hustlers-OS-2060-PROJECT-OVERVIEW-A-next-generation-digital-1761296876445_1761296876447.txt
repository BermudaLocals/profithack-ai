=== DATABASE SCHEMA ===

Users Table:
- id (primary key)
- email (unique)
- password_hash
- subscription_tier (enum: free/creator/pro)
- credits (integer)
- is_verified (boolean)
- created_at (timestamp)
- last_active (timestamp)

Projects Table:
- id (primary key)
- user_id (foreign key)
- name (string)
- description (text)
- code (text)
- is_public (boolean)
- created_at (timestamp)
- updated_at (timestamp)

Videos Table:
- id (primary key)
- user_id (foreign key)
- title (string)
- description (text)
- video_url (string)
- thumbnail_url (string)
- views (integer)
- gifts_earned (decimal)
- created_at (timestamp)

Gifts Table:
- id (primary key)
- sender_id (foreign key)
- receiver_id (foreign key)
- video_id (foreign key)
- amount (decimal)
- message (text)
- created_at (timestamp)

Transactions Table:
- id (primary key)
- user_id (foreign key)
- type (enum: subscription/gift/withdrawal)
- amount (decimal)
- status (enum: pending/completed)
- created_at (timestamp)
