More products
Webhooks
Get notifications for events on TikTok with lightweight subscriptions integrated to your app or website. Receive data in real-time without making API requests.
Data Portability API
Allow users to transfer from TikTok to your app. Available only to TikTok users in the EEA and UK.
Scopes
Request access to scopes so your app can request user authorization for data, query TikTok's APIs, and perform different operations.
Green Screen Kit
Share images or videos from your app directly to TikTok to use as a green screen background.
Display API
Display TikTok creators' recent or self-selected videos and their profile information on your app.
Research Tools
Access TikTok's public data on user profiles, videos, comments, and more. Available only to approved researchers.
Commercial Content API
Perform customized searches on ads and other commercial content data. Available only to approved researchers.
